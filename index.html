Projects

30-Year Ice Atlas Revision

The 30-Year Ice Atlas represents the changes in Ice in the Canadian Waters over a 30-Year Period with respect to its median concentration, predominant type, frequency of presence and more. Every decade, the images are recreated with updated data.

For this year's Atlas, the goal of this project was to not only create new charts with updated data but to also re-implement it from bottom-up using more modern open source tools like QGIS and Python libraries like OSGEO.

For example, a Median Concentration of Ice image is generated by first analyzing all ingested data from the past 30 years, computing the median concentration by Polygon intersections and further creating the final image.
